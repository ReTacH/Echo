Here is what we've found out about Echo Chamber:

We asked the author of [1] for the dataset they used, and produced our own production scores and consumption scores after filtering the tweets with known user data and with quoted domains.

We then checked the clusterings of the polarity scores we produced using the three types of graphs below:

With the heatmap, we can see that there are two clear clusterings! This indicates that the users we investigated are highly polarized.

Network Analysis

We also created a network of twitter users in the dataset with NetworkX, used the modularity method to separate possible communities, and plotted it using Gephi.

It's actually quite similar to the clustering we created using polarity scores!

Takeaway: there's a strong correlation between the Twitter social network structures and derived polarity scores!
